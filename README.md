# MacOS Init

Personal script for first-time MacOS init. Installs CLI basics, coding agents, and programming languages. Also installs my baseline Vim configuration. 

```
chmod u+x start.sh
./start.sh
```

## System Configuration

### System Settings

Set scroll direction to my preference and set fast key-repeat for Vim usefulness.

**Trackpad / Mouse**
1. Go to **System Settings > Trackpad** (or Mouse) > **Scroll & Zoom**.
2. Toggle **Natural scrolling** off.

**Keyboard**
1. Go to **System Settings > Keyboard**.
2. Set **Key repeat rate** to **Fast** (Move slider to the far right).
3. Set **Delay until repeat** to **Short** (Move slider to the far right).
4. Under **Text Input**, click **Edit...**.
5. Disable **Correct spelling automatically**.
6. Disable **Capitalize words automatically**.

### Finder Preferences

1. Open **Finder**.
2. In the menu bar, go to **Finder > Settings... > General**.
3. Under **Show these items on the desktop**, check **Hard disks** and **External disks**.
4. Set **New Finder windows show** to **Home** directory.
5. In the menu bar, go to **View**.
6. Select **Show Path Bar**.

### Screenshots

Placing screenshots under a `Screenshots` folder instead of directly on desktop will help keeping it tidy.

1. Press `Command + Shift + 5` to open the Screenshot utility.
2. Click **Options**.
3. Under **Save to**, select **Other Location...** and choose `~/Desktop/Screenshots`.
